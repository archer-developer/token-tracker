# Описание экранов приложения

## 1. Портфель (Portfolio) — Дашборд

### Расположение

Главный экран приложения, открывается по умолчанию.

### Что видно

#### Сверху: Меню навигации

- Логотип/название приложения
- Ссылки на все разделы: Портфель, Инструменты, Календарь, Журнал, Настройки
- На мобильной версии: нижняя панель с иконками

#### Основной контент: Блок метрик (2×3 таблица)

**Верхний ряд:**

```
┌─────────────────────────┐  ┌─────────────────────────┐
│  Инвестировано          │  │  Осталось получить      │
│  -17,500 USD            │  │  3,000 USD + 8,500 USD  │
└─────────────────────────┘  └─────────────────────────┘
```

**Средний ряд:**

```
┌─────────────────────────┐  ┌─────────────────────────┐
│  Получено доход         │  │  Текущие убытки         │
│  +700 USD               │  │  -3,000 USD             │
└─────────────────────────┘  └─────────────────────────┘
```

**Нижний ряд:**

```
┌─────────────────────────┐  ┌─────────────────────────┐
│  P&L (Прибыль/убыток)   │  │  XIRR (Доходность)      │
│  -19,800 USD            │  │  5.2% годовых          │
└─────────────────────────┘  └─────────────────────────┘
```

#### График портфеля

Линейный график с двумя линиями:

- **Синяя** — история P&L (с начала до сегодня)
- **Оранжевая** — прогноз P&L (с сегодня до конца инвестиции)

Ось X: месяцы/даты
Ось Y: сумма в выбранной валюте

#### Все инструменты (список)

Карточки по 2 в ряд (десктоп) с краткой информацией:

- Название и статус (зеленый/синий/красный)
- Ставка купона
- Дата начала / Дата конца
- Статус запланированных выплат (N запланировано, M выплачено)

## 3. Детали инструмента

### Верхняя панель

- ← Кнопка "Назад"
- Название инструмента
- Кнопка "Редактировать" (✏️)
- Кнопка "Удалить" (🗑️)

### Основная информация

Карточка с условиями облигации:

- Платформа: Fainex
- Валюта: USD
- Ставка купона: 11.0% годовых
- Цена токена: 100 USD
- Периодичность выплат: Ежемесячно
- Дата начала: 09.01.2025
- Дата погашения: 09.01.2026

### Статус и действия

**Если статус "Активный":**

- [Отметить как созревший]
- [Зарегистрировать дефолт]
- [Продать]

**Если статус "Дефолт":**

- Поле для указания ожидаемого возврата (%)
- [Обновить ожидаемый возврат]

### Метрики инструмента (2×2 таблица)

```
┌──────────────────┐  ┌──────────────────┐
│ Инвестировано    │  │ Осталось получить│
│ 15,100 USD       │  │ 2,000 USD        │
│                  │  │ + 5,000 USD      │
└──────────────────┘  └──────────────────┘

┌──────────────────┐  ┌──────────────────┐
│ Получено доход   │  │ P&L инструмента  │
│ 600 USD          │  │ -11,900 USD      │
└──────────────────┘  └──────────────────┘
```

### Лоты покупки (Purchase Lots)

Таблица всех покупок этого инструмента:

| Дата       | Кол-во токенов | Цена за токен | Сумма      | Действия |
| ---------- | -------------- | ------------- | ---------- | -------- |
| 09.01.2025 | 100            | 100 USD       | 10,000 USD | ✏️ 🗑️    |
| 26.01.2025 | 50             | 102 USD       | 5,100 USD  | ✏️ 🗑️    |

**Итого инвестировано: 15,100 USD**

На мобильной версии кнопки редактирования/удаления стилизованы как кнопки (не просто иконки).

### Выплаты (Payments)

#### Вкладки

- **Запланировано** — выплаты, которые еще не произошли
- **История** — выплаты уже полученные или пропущенные

#### Запланированные выплаты

Карточки по одной:

```
🟡 Купон
09.01.2025 – 09.02.2025
Ожидаемая сумма: 91.67 USD

[Отметить как выплаченную] [Отметить как пропущенную]
```

При клике "Отметить как выплаченную":

1. Откроется модальное окно
2. Поле для ввода "Фактическая сумма" (по умолчанию = Ожидаемая)
3. Поле "Дата выплаты"
4. [Отмена] [Сохранить]

При клике "Отметить как пропущенную":

1. Появится диалог подтверждения "Вы уверены?"
2. [Отмена] [Подтвердить]

#### История выплат

Карточки с информацией о полученных/пропущенных выплатах:

```
🟢 Купон (Выплачено)
09.01.2025 – 09.02.2025
Ожидаемая сумма: 91.67 USD
Фактическая сумма: 91.67 USD ✓
```

## 4. Календарь выплат (Calendar)

### Структура

Месячный календарь с выплатами:

- Две вкладки: "По инструментам" | "По типам"
- Левая/правая стрелка для навигации по месяцам

### Вкладка "По инструментам"

Все выплаты сгруппированы по названиям облигаций:

```
Январь 2025
═════════════════════════════════════

🟢 GURMINA.USD.2024.01 (Активная)
  15 января — Купон 91.67 USD (Выплачено) ✓

🔵 COMPASS.2024.Q4 (Созревшая)
  20 января — Погашение 5,000 USD (Выплачено) ✓
```

### Вкладка "По типам"

Выплаты сгруппированы по типам (Купоны, Погашение, Возвраты):

```
Январь 2025
═════════════════════════════════════

💚 КУПОНЫ (150 USD в месяц)
  GURMINA — 91.67 USD (Выплачено) ✓
  COMPASS — 58.33 USD (Выплачено) ✓

💙 ПОГАШЕНИЕ (5,000 USD)
  COMPASS — 5,000 USD (Выплачено) ✓
```

## 5. Журнал денежных потоков (Ledger)

### Таблица со всеми операциями

| Дата       | Инструмент      | Тип       | Сумма       | Примечание |
| ---------- | --------------- | --------- | ----------- | ---------- |
| 09.01.2025 | GURMINA.USD...  | Покупка   | -10,000 USD | Лот 1      |
| 26.01.2025 | GURMINA.USD...  | Покупка   | -5,100 USD  | Лот 2      |
| 09.02.2025 | GURMINA.USD...  | Купон     | +91.67 USD  | Выплачено  |
| 26.02.2025 | COMPASS.2024... | Покупка   | -7,500 USD  | Лот 1      |
| 20.03.2025 | COMPASS.2024... | Погашение | +7,500 USD  | Выплачено  |

**Типы операций:**

- **Покупка** (Purchase) — красная иконка, минус сумма
- **Купон** (Coupon) — синяя иконка, плюс сумма
- **Погашение** (Redemption) — зеленая иконка, плюс сумма
- **Возврат после дефолта** (Recovery) — оранжевая иконка, плюс сумма
- **Продажа** (Sale) — фиолетовая иконка, плюс сумма

### Фильтры

- По инструменту
- По типу операции
- По диапазону дат

## 6. Настройки (Settings)

### Общие (General)

#### Тема (Theme)

Три кнопки: ☀️ Светлая | 🌙 Темная | 🖥️ Система

#### Язык (Language)

Две кнопки: Русский | Беларуская

#### Валюта для отображения (Base Currency)

Три кнопки: BYN | USD | EUR

Все суммы в приложении пересчитываются в выбранную валюту.

#### Скрывать суммы (Hide Amounts)

Галочка для скрытия всех денежных сумм (режим "для показа на публику")

#### Показывать нулевые выплаты (Show Zero Payments)

Галочка для отображения выплат со сбросовой или нулевой суммой

#### Режим презентации (Presentation Mode)

Галочка — загружает демо-данные (не используется для реальных инвестиций)

### Курсы валют (Exchange Rates)

#### Текущие курсы

Таблица с текущими курсами:

- 1 USD = X.XXXX BYN
- 1 EUR = X.XXXX BYN

Дата обновления: 06.06.2026

#### Обновить курсы

Кнопка [🔄 Обновить] — вручную получит актуальные курсы с сервера NBRB

**Автоматическое обновление:** Приложение проверяет курсы дважды в день:

- До полудня (00:00)
- После полудня (12:00)

Если вы откроете приложение после этого времени, курсы обновятся автоматически.

### LLM интеграция (LLM Integration)

#### Настройка подключения

Три поля для ввода:

1. **Base URL** — адрес сервера LLM (например, https://api.openai.com/v1)
2. **API Key** — секретный ключ доступа (маскируется звездочками)
3. **Model** — название модели (например, gpt-4o-mini)

#### Тестирование

Кнопка [✨ Тест] проверит соединение с LLM

### Резервная копия (Backup)

#### Экспортировать

Кнопка [⬇️ Экспортировать] загружает файл `tokens-tracker-backup-ДАТА.json` со всеми данными.

#### Импортировать

Кнопка [⬆️ Импортировать] позволяет загрузить JSON файл и восстановить все данные.

**Внимание!** При импорте все текущие данные будут заменены.

## Навигация между экранами

```
Портфель
   ↓
   ├─→ Клик на инструмент → Детали инструмента
   │                            ├─→ [Редактировать] → Форма инструмента
   │                            ├─→ [Лоты] → Добавить/Редактировать лот
   │                            └─→ [Выплаты] → Отметить выплату
   │
   ├─→ Инструменты → Список всех инструментов
   │                    ├─→ Клик на строку → Детали инструмента
   │                    └─→ [Добавить] → Форма инструмента (новый)
   │
   ├─→ Календарь → Месячный календарь выплат
   │
   ├─→ Журнал → История всех денежных потоков
   │
   └─→ Настройки → Конфигурация приложения
```

## Модальные окна и диалоги

### Добавление/Редактирование инструмента

Форма с полями:

- Название
- Платформа (выбор)
- Валюта (выбор)
- Ставка купона
- Стоимость токена
- Дата начала
- Дата погашения
- Периодичность выплат
- Ссылка на White Paper
- Кнопка [Извлечь из PDF] (если настроен LLM)

**Кнопки:** [Отмена] [Сохранить]

### Добавление/Редактирование лота

Форма с полями:

- Дата покупки
- Количество токенов
- Цена за токен
- (Автоматически: Сумма = Кол-во × Цена)
- Примечание (опционально)

**Кнопки:** [Отмена] [Сохранить]

### Отметить выплату как выплаченную

Модаль с полями:

- Фактическая сумма (число)
- Дата выплаты (календарь)

**Кнопки:** [Отмена] [Сохранить]

### Отметить выплату как пропущенную

Диалог подтверждения с сообщением:
"Вы уверены, что хотите отметить эту выплату как пропущенную?"

**Кнопки:** [Отмена] [Подтвердить]

### Удаление инструмента / лота

Диалог подтверждения с информацией:

- Что удаляется
- Дата/сумма
- Предупреждение "Это действие нельзя отменить"

**Кнопки:** [Отмена] [Удалить]

## Мобильная версия

### Отличия от десктопа

1. **Навигация:** Нижняя панель с 5 иконками (вместо верхнего меню)
2. **Таблицы:** Преобразованы в карточки (одна карточка = одна строка таблицы)
3. **Кнопки:** Текст скрывается, видны только иконки (для экономии места)
4. **Графики:** Интерактивные (можно прокручивать горизонтально)
5. **Шрифты:** Мельче, чем на десктопе
6. **Расстояния:** Меньше padding/margin

### Адаптивные точки (Breakpoints)

- **< 640px** — мобильная версия (sm)
- **640px – 1024px** — планшет (md, lg)
- **> 1024px** — десктоп (xl)

---

## Сочетания клавиш (Keyboard Shortcuts)

- **Enter** — отправить форму (в модальных окнах)
- **Esc** — закрыть модаль/диалог
- **Tab** — переместиться на следующее поле

На мобильных устройствах стандартные жесты:

- **Свайп влево/вправо** — навигация между месяцами (календарь)
- **Долгое нажатие** — копирование суммы (в некоторых местах)
